## How to run locally

Requirements: Node.js and npm (or pnpm/yarn).

```sh
# clone the repository
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>

# install dependencies
npm install

# start development server
npm run dev
```

## Technologies used

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

You can deploy this project to platforms like Vercel, Netlify or GitHub Pages. Build the app with `npm run build` and follow your host's deployment instructions.

If you want a custom Open Graph image for nicer link previews, put an image in `public/` (e.g. `public/opengraph.png`) and update `index.html`'s `og:image` and `twitter:image` meta tags.
